# fabric-cicd

Microsoft Fabric CI/CD scaffold for Azure DevOps.

The implementation target is:

- Azure DevOps Repos for Fabric source control.
- Azure Pipelines for CI, release, approvals, and promotion.
- Terraform for Fabric workspace setup, Git integration, Azure operational resources, and Azure DevOps pipeline definitions.
